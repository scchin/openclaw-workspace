# Compliance & Ethics Roadmap for ILIB

## 1. Anti-Doping (WADA M1.3)
*   **Constraint**: ILIB falls under "Manipulation of Blood".
*   **Paradox**: Prohibited not because it's harmful, but because it provides an unfair physiological advantage (ergogenic gain).
*   **Requirement**: Competitive athletes MUST have a Therapeutic Use Exemption (TUE).

## 2. Regulatory Pathways
*   **FDA**: Class II (510(k)) or Class III (PMA).
*   **EMA**: Class IIb or III under EU MDR.
*   **Rule**: Use of non-certified "grey-market" hardware is prohibited.

## 3. Legal Boundaries
*   **Hospital Wall**: Must be performed in licensed medical facilities.
*   **License Lock**: Only certified physicians/phlebotomists may perform insertion.
*   **Consent**: Documented informed consent for invasive risks is mandatory.
