# Expert Registry (專家能力圖譜)

## 📋 制度說明
本文件為 `Dynamic Expert Dispatcher (DED)` 的核心數據庫。所有被派遣的專家必須在此登記其「認知 DNA」。

### 登記維度定義
- **專家名稱 (Expert Name)**: 唯一識別符。
- **核心專長 (Core Expertise)**: 負責處理的具體領域。
- **認知原型 (Cognitive Archetype)**: 思考風格（例如：第一原理、對立思考、數據驅動）。
- **邏輯模式 (Logic Pattern)**: 執行分析時遵循的特定步驟。
- **對標對象 (Reference)**: 模擬的現實人物或專業標準。
