### browser tool 架構限制（已確認）

**問題核心：`screenshot` / `snapshot` 綁定 `openclaw` profile（port 18800），無法操作 `chrome` relay tabs（port 18792）**

- `profile="chrome"` 時：`open` → 成功開新分頁並返回正確 WS URL；`tabs` → 成功列出 relay tabs
- 但 `screenshot` / `snapshot` / `act` → 內部 CDP 客戶端固定連接 `ws://127.0.0.1:18800/devtools/page/...`，無法改用 `ws://127.0.0.1:18792/cdp?token=...`
- 因此 relay tab (`targetId` 格式不同) 無法被 screenshot/snapshot 操作

- `npx playwright` → playwright package not in /tmp
- `tesseract` → system binary not found
