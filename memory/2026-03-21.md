### CDP 讀取技術突破（已驗證可行）
經過大量 Debug，確認以下方法繞過 Google Maps 反爬蟲機制：

1. **每人消費讀取**：直接用 `innerText` 解析 `"$1–200 per person, Reported by 71"`
2. **點擊按鈕**：`querySelector` 無法找到 shadow DOM 按鈕，需用 `innerText` 找 div → `scrollIntoView` → `element.click()`
3. **價格分布讀取**：點擊後popup含TD元素（`$1–200`、`$200–400`、`$400–600`），但count為空
4. **Teal Bar 技術**：popup內的分布柱是 `rgb(0,123,139)` 的 div，寬度 ÷ 容器寬度 = 比例，人數 = 比例 × 71（總回報人數）
5. **服務/Menu**：aria-label + link href

- `browser` tool 是 AI agent 工具，Python script 無法直接呼叫

- **正確分工**：`browser` tool 啟動瀏覽器 → AI agent 呼叫 `query.py` 執行 CDP
